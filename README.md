 ### 3. Most Frequent Word Counter

A beginner Python project that finds the most frequent word in a text using dictionary.

Built as part of my NLP learning journey — Month 1: Python & Text Processing Foundation.

## How It Works

1. Split text into a list of words
2. Count word frequency using dictionary
3. Find the most frequent word using max()

## Concepts Used

- Dictionary — store word-frequency pairs
- .get(key, default) — safely count words
- max() — find the most frequent word
- .split() — split text into words

## How to Run

python most_frequent_words.py
