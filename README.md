# Word Counter & Character Counter

Mini Python projects built as part of my journey transitioning from *English Linguistics* into *Computational Linguistics / NLP*.

## About
These are foundational text-processing scripts that demonstrate basic string manipulation in Python — the building blocks for more advanced Natural Language Processing (NLP) tasks.

## Projects

### 1. Word Counter
Counts the number of words in a given text input.

*Concepts used:*
- input()
- .split()
- len()

### 2. Character Counter
Counts the number of characters in a given text input.

*Concepts used:*
- input()
- len()
- String methods (.capitalize(), .upper(), .lower())

## How to Run
```bash
python word_counter.py
python character_counter.py
